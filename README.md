## WHATS INCLUDED

- React 16 (Obviously)
- react-waterfall (For state management)
- Eslint (airbnb style)
- Prettier

- Jest
- Enzyme
- Webpack
- Travis CI and Coveralls integration
- VSCode settings for Chrome Debugging/Prettier etc

## GET IT

```
git clone <url> .
rm -rf ./git
git init
```

## USE IT

```sh
yarn start
yarn build
yarn test
yarn coverage
yarn lint
yarn flow
```
